# Data Integration Platform

Descripción: Plataforma para integración de datos, con backend, Docker y pruebas.

## Instalación

1. Clona el repo.
2. Crea un entorno virtual: `python -m venv .venv`
3. Activa: `.venv\Scripts\activate` (Windows)
4. Instala dependencias: `pip install -r requirements.txt` (si tienes uno)

## Uso

- Backend: Ejecuta desde `backend/`
- Docker: Usa `docker-compose up` desde `docker/`

## Tests

Ejecuta `pytest` en `tests/`
